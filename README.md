# Joel's Portfolio

> A minimal, dark-themed portfolio website built for speed and aesthetics. Showcasing my journey as an AI Engineer and Creative Developer.

## Features

- **Modern Dark Aesthetic**: Deep greens and acid lime accents for a hacker/tech vibe.
- **Fast & Lightweight**: Built with semantic HTML5, optimized CSS3, and vanilla JavaScript. No heavy frameworks.
- **Fully Responsive**: Looks great on mobile, tablet, and desktop (Mobile-First approach).
- **Fluid Animations**: Apple-style scroll reveals and staggered grid animations using IntersectionObserver.
- **Accessible**: Semantic labels, proper contrast, and keyboard navigation support.

## Tech Stack

- **Frontend**: HTML5, CSS3 (Variables, Flexbox, Grid), JavaScript (ES6+)
- **Fonts**: [Inter](https://fonts.google.com/specimen/Inter) (Body) & [Space Mono](https://fonts.google.com/specimen/Space+Mono) (Headers/Code)
- **Deployment**: GitHub Pages

## Getting Started

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/git-itsjoel/git-itsjoel.github.io.git
   ```
2. **Navigate to the directory**
   ```bash
   cd git-itsjoel.github.io
   ```
3. **Open `index.html`** in your browser.
   - Or use a live server (e.g., VS Code Live Server extension).
